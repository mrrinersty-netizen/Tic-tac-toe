# Tic-Tac-Toe game WinForms

Простое игровое приложение «Крестики-нолики», разработанное на языке C# с использованием фреймворка Windows Forms.

## Возможности

- Графический интерфейс
- Игровое поле 3×3
- Управление мышью
- Проверка победы и ничьей
- Сохранение игры в файл
- Загрузка сохранённой игры
- Начало новой игры

## Используемые технологии

- C#
- Windows Forms
- .NET

## Запуск проекта

1. Открыть проект в Visual Studio
2. Собрать проект
3. Запустить приложение

## Сохранение игры

Состояние игры сохраняется в файл:

```text
save.txt
```

Файл содержит:
- текущего игрока
- состояние игрового поля

## Структура проекта

```text
Form1.cs
Form1.Designer.cs
Program.cs
save.txt
```
